MEASUREMENT UNIT



cm : centimeter

mm : millimeter

in : inches (1in = 96px = 2.54cm)

px : pixels

pt : points (1pt = 1/72in)

pc : picas (1pc = 12pt = 1/6in)

em : Relative to the font-size of the element (2em means 2 times the size of the current font)

ex : Relative to the x-height of the current font (rarely used) (1ex = 6px)

ch : Relative to the width of the "0" (zero) (basically 1ch = 8px)

rem : Relative to font-size of the root element ( 1rem = 16px)

vw : Relative to 1% of the width of the viewport*

vh : Relative to 1% of the height of the viewport*

vmin : Relative to 1% of viewport's* smaller dimension

vmax : Relative to 1% of viewport's* larger dimension

% : Relative to the parent element

BASIC



<!DOCTYPE> : Defines the document type

<html> : Define HTML document

<head> : Contains metadata/information for the document

<title> : Defines a title for the document

<h1> to <h6> : Defines HTML headings.

<body> : Defines the document's body

<p> : Defines a paragraph

<br> : Insert single line break

<hr> : Defines a thematic change in the content. basically add horizontal line

<!--.....--> : Use to add comment

FORMATTING



<abbr> : Defines an abbreviation or an acronym [ex: hello ]

<address> : Defines contact information for the author/owner of a document/article [ex:

surat , gujarat
]

<b> : Defines bold text [ex: hello]

<bdi> : Isolates a part of text that might be formatted in a different direction from other text outside it [note: not useful ]

<bdo dir=" "> : Overrides the current text direction. value of dir defines the direction of string. dir="ltr" define left to right & dir="rtl" define right to left [ex: ltr: hello & rtl:hello ]

<blockquote> : Defines a section that is quoted from another source. start from new line [ex:

hello
]

<cite> : Defines the title of a work [ex: hello ]

<code> : Defines a piece of computer code [ex: hello ]

<del> : Defines text that has been deleted from a document [ex: hello ]

<dfn> : Specifies a term that is going to be defined within the content [ex: hello ]

<em> : Defines emphasized text [em: hello ]

<i> : Defines a part of text in an alternate voice or mood [ex: hello ]

<ins> : Defines a text that has been inserted into a document [ex: hello ]

<kbd> : Defines keyboard input [ex: hello ]

<mark> : Defines marked/highlighted text [ex: hello]

<meter> : Defines a scalar measurement within a known range (a gauge)

<pre> : Defines preformatted text

<progress> : Represents the progress of a task

<q> : Defines a short quotation

<rp> : Defines what to show in browsers that do not support ruby annotations

<rt> : Defines an explanation/pronunciation of characters (for East Asian typography)

<ruby> : Defines a ruby annotation (for East Asian typography)

<s> : Defines text that is no longer correct .[ similar to del tag]

<samp> : Defines sample output from a computer program

<small> : Defines smaller text

<storng> : Defines important text

<sub> : Defines subscripted text

<sup> : Defines superscripted text

<time> : Defines a specific time (or datetime)

<u> : Defines some text that is unarticulated and styled differently from normal text
